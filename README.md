# Movia ServiceStack - Countdown Test Client

This is a simple test client for the Countdown Service, part of Movia ServiceStack.

The Countdown Service exposes departure and deviation data, primarily for use in the countdown modules installed at bus stops.

## Build
To build the Test Client run the included build PowerShell-script:
```
> build.ps1
```

This will produce an executable file in the ``dist`` folder.
